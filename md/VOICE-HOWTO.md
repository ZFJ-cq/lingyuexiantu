# 灵月仙途 - 语音朗读使用指南

## 🔊 为什么没有声音？

### 原因说明
1. **语音朗读默认关闭** - 需要手动点击开启
2. **浏览器需要用户交互** - 第一次点击才能激活语音功能
3. **系统音量可能关闭** - 请检查设备音量

---

## 📱 如何开启语音朗读

### 步骤 1: 打开页面
访问：`http://localhost:8080/intro.html`

### 步骤 2: 点击语音按钮
在屏幕**左上角**找到语音控制按钮：
- 🔇 (灰色) - 语音关闭状态
- 🔊 (金色) - 语音开启状态

**点击它！**图标会从 🔇 变为 🔊

### 步骤 3: 开始朗读
开启后会自动朗读当前场景的文字：
- **第一幕**: "鸿蒙纪元三千载，天裂地崩九气散..."
- **第二幕**: "传闻飞升者，皆入三千灵域迷局..."
- **第三幕**: "路在脚下，道在心中..."

## 🎤 语音功能详解

### 语音控制按钮位置
```
┌─────────────────────────┐
│ 🔊 🔇        跳过动画  │ ← 左上角
└─────────────────────────┘
  ↑
  语音按钮
```

### 按钮状态
| 图标 | 颜色 | 状态 | 说明 |
|------|------|------|------|
| 🔇 | 灰色 | 关闭 | 语音未启用 |
| 🔊 | 金色 | 开启 | 语音正在朗读 |

### 操作方式
1. **点击一次** 🔇 → 🔊 : 开启语音，开始朗读
2. **点击一次** 🔊 → 🔇 : 关闭语音，停止朗读
3. **场景切换** : 自动继续朗读新场景

## 🔧 故障排查

### 问题 1: 点击按钮没反应
**可能原因**:
1. 浏览器不支持 Web Speech API
2. 页面还未完全加载

**解决方法**:
1. 打开浏览器控制台（F12）
2. 查看是否有错误信息
3. 刷新页面重试

**检测代码**（在控制台运行）:
```javascript
if ('speechSynthesis' in window) {
  console.log('✅ 支持语音功能');
} else {
  console.log('❌ 不支持语音功能');
}
```

### 问题 2: 没有声音输出
**可能原因**:
1. 系统静音
2. 浏览器音量关闭
3. 设备扬声器关闭

**解决方法**:
1. ✅ 检查系统音量（右上角喇叭图标）
2. ✅ 检查浏览器音量（浏览器设置）
3. ✅ 检查设备是否静音

### 问题 3: 语音是机器音/很难听
**原因**: 使用了系统默认语音

**解决方法**:
1. 打开控制台（F12）
2. 查看语音名称
3. 推荐语音：
   - **Safari**: "Ting-Ting" 或 "Sinji"
   - **Chrome**: "Hui Hui" 或 "Kangkang"

**查看可用语音**（在控制台运行）:
```javascript
speechSynthesis.getVoices().forEach(voice => {
  console.log(voice.name, voice.lang);
});
```

### 问题 4: 朗读中断/不继续
**可能原因**:
1. 场景切换时被打断
2. 页面失去焦点

**解决方法**:
- 代码已自动处理，转场后 1 秒会自动继续朗读
- 确保浏览器窗口在前台

## 📱 不同设备的使用

### iPhone/iPad (Safari)
✅ **最佳体验**
- 语音质量最高（苹果原生 TTS）
- 点击任意位置激活语音
- 推荐佩戴耳机

### Android (Chrome)
✅ **良好体验**
- 语音质量取决于系统
- 部分手机需要安装 Google TTS
- 建议调大音量

### 电脑浏览器
✅ **完美支持**
- Chrome/Edge/Safari 都支持
- Firefox 支持但质量一般
- 推荐 Chrome 或 Edge

### 微信内置浏览器
⚠️ **部分支持**
- 依赖系统 TTS 引擎
- 可能没有中文语音
- 建议用手机浏览器打开

## 🎵 语音 + BGM 配合

### 推荐配置
1. **仅语音**（推荐首次体验）
   - 点击 🔇 开启语音
   - 专注听文案内容

2. **语音 + BGM**（推荐二次体验）
   - 点击 🎵 开启 BGM
   - 点击 🔇 开启语音
   - BGM 音量 50%，语音 100%

3. **仅 BGM**
   - 沉浸式体验
   - 自己阅读文字

### 音量平衡
```
BGM:    50-60%  (不盖过语音)
语音：  100%    (清晰可听)
音效：  70-80%  (适中)
```
