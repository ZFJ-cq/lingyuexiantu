# 🚨 后端服务已停止！

## 问题

代理服务器返回 502 错误，因为后端 Java 服务已停止运行。

## ✅ 解决方案

### 使用 IDEA 启动后端（推荐）

1. **打开 IntelliJ IDEA**
2. **打开项目**：`/Users/macbook/前端项目/灵月仙途/lingyuexiantu-server`
3. **等待 Maven 同步完成**
4. **找到主类**：`com.lingyue.LingyuexiantuServerApplication`
5. **右键点击 → Run 'LingyuexiantuServerApplication'**

或者：
- 在 IDEA 右侧找到 Maven 面板
- 展开 `lingyuexiantu-server` → `Plugins` → `spring-boot-maven-plugin`
- 双击 `spring-boot:run`

### 启动成功的标志

在 IDEA 控制台看到：
```

Tomcat started on port 8088 (http) with context path '/api'
```
