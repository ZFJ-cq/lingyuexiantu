## 二、榜单系统（进行中 🔄）

### 2.1 功能描述
- 全服排行榜（战力榜/修为榜）
- 实时刷新数据
- 前 10 名展示特殊头像框
- 前三名特殊标识

### 2.2 UI 设计
- **头像框**：前 10 名金色边框 + 光晕特效
- **榜首标识**：第一名显示"榜首"标签
- **排名颜色**：
  - 第 1 名：金色 (#FFD700)
  - 第 2 名：银色 (#C0C0C0)
  - 第 3 名：铜色 (#CD7F32)
  - 第 4-10 名：淡金色 (#E6C749)

### 2.3 数据格式
```json
{
  "code": 200,
  "data": [
    {
      "rank": 1,
      "roleId": 1,
      "roleName": "张三",
      "avatar": "url",
      "combatPower": 10000,
      "realm": "金丹期",
      "level": 50,
      "server": "一服"
    }
  ]
}
```

### 2.4 待完成
- ❌ 后端 LeaderboardController 重构
- ❌ 战力/修为排行榜接口
- ❌ 实时刷新机制（WebSocket 或定时轮询）

  id BIGINT PRIMARY KEY,

CREATE TABLE friend_request (

## 开发规范

### API 格式
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 错误码
- 200: 成功
- 400: 参数错误
- 401: 未授权
- 403: 禁止访问
- 404: 资源不存在
- 500: 服务器错误
