### 2. 静态资源拦截问题 ✅
**问题：** 所有 API 请求都被当作静态资源处理，返回 500 错误
**原因：** Spring Boot 默认启用了静态资源映射
**解决方案：** 在 `application.yml` 中添加配置：
```yaml
spring:
  web:
    resources:
      add-mappings: false  # 禁用默认静态资源映射
```

### 3. API 路径错误 ✅
**问题：** 前端调用的 API 路径与后端不匹配
