# 仙石弹窗样式优化

## 修改内容

已将仙石/灵石的弹窗样式修改为 **2*3 小正方形模块** 布局。

## 功能特性

### 1. 智能识别
- 自动识别类型为"灵石"或"仙石"的资产
- 自动识别名称包含"灵石"或"仙石"的资产
- 其他资产类型保持原有弹窗样式

### 2. 2*3 网格布局
弹窗内显示 6 个小正方形模块，每个模块包含：
- **下品灵石** 💎
- **中品灵石** 💠
- **上品灵石** 🔮
- **极品灵石** ✨
- **仙石** 🌟
- **神石** ⭐

### 3. 视觉效果
- 每个模块为正方形（aspect-ratio: 1）
- 渐变背景 + 金色边框
- 悬停时有发光和上浮效果
- 图标带有阴影特效

### 4. 数值计算
根据资产总数量自动换算各品阶灵石：
- 下品灵石 = 总数 / 1,000,000
- 中品灵石 = (总数 / 10,000) % 100
- 上品灵石 = (总数 / 100) % 100
- 极品灵石 = 总数 % 100
- 仙石 = 总数 / 100,000,000
- 神石 = 总数 / 10,000,000,000
