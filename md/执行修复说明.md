# 📋 角色数据修复 - 操作指南

## 方法 1：在浏览器控制台执行（推荐）

### 步骤：

1. **打开游戏页面**
   - 访问：http://127.0.0.1:5502/index.html
   - 或者您正在使用的游戏页面

2. **打开浏览器控制台**
   - Mac: 按 `Cmd + Option + J`
   - Windows: 按 `F12` 然后切换到 "Console" 标签

3. **复制并粘贴以下代码**，然后按回车：

```javascript
// 执行修复
fetch('http://localhost:8088/api/fix/role/1', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' }
})
.then(res => res.json())
.then(data => {
  console.log('修复结果:', data);
  if (data.code === 200) {
    console.log('✅ 修复成功！');
    console.log('角色数据:', data.data);
    setTimeout(() => location.reload(), 2000);
  }
});
```

4. **等待 2 秒**，页面会自动刷新

5. **验证修复结果**：
   - 境界应该显示"炼气一层"
   - 角色数据应该完整

## 方法 3：直接在 PHPMyAdmin 执行 SQL

如果您使用 PHPMyAdmin 管理数据库：

1. **打开 PHPMyAdmin**
   - 访问：http://localhost/phpmyadmin

2. **选择数据库**：`lingyuexiantu`

3. **执行 SQL**：
```sql
UPDATE game_role 
SET 
    gender = 1,
    realm = '炼气一层',
    level = 1
WHERE id = 1 AND (gender IS NULL OR realm IS NULL OR level IS NULL);
```

4. **刷新游戏页面**

## ❓ 常见问题

### Q1: 提示"网络错误"？
**解决**：确保后端服务器正在运行
```bash
# 检查端口 8088 是否被占用
lsof -i:8088
```

### Q2: 修复后还是显示 null？
**解决**：清除浏览器缓存
- Mac: `Cmd + Shift + R`
- Windows: `Ctrl + Shift + R`

### Q3: 提示"角色不存在"？
**解决**：确认角色 ID 是否正确
```javascript
// 查看当前角色 ID
console.log('当前角色 ID:', localStorage.getItem('currentRoleId'));
```
