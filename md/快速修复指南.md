## 🐛 常见问题快速解决

### 问题 1: MySQL 服务未启动
```bash
# macOS
brew services start mysql

# 或者
mysql.server start

# 检查状态
mysql.server status
```

### 问题 2: 数据库不存在
```bash
# 创建数据库
mysql -h localhost -P 3306 -u root -p -e "CREATE DATABASE IF NOT EXISTS lingyuexiantu DEFAULT CHARSET utf8mb4;"
```

### 问题 3: 权限不足
```bash
# 授权 (在 MySQL 中执行)
GRANT ALL PRIVILEGES ON lingyuexiantu.* TO 'root'@'localhost';
FLUSH PRIVILEGES;
```

### 问题 4: 表不存在
```bash
# 先执行基础表创建
mysql -h localhost -P 3306 -u root -p lingyuexiantu < mock_data.sql
```

### 问题 5: 字段已存在警告
这是**正常提示**,表示字段已经修复过，可以忽略。
