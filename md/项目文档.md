# 《灵月仙途》项目文档

## 1. 项目概述

《灵月仙途》是一款基于Web的修仙题材游戏，采用前后端分离架构，从原型阶段升级为生产就绪阶段。项目实现了用户认证、角色管理、背包系统、好友系统、商城系统、任务系统、邮件系统、活动系统和成就系统等核心功能。

## 2. 技术栈

### 前端技术
- HTML5 + CSS3 + JavaScript
- 响应式设计，支持移动端
- 原生JavaScript实现API调用和页面交互
- 模块化组织代码

### 后端技术
- Spring Boot 3.2.0
- Spring Security
- JPA (Java Persistence API)
- MySQL数据库
- RESTful API设计

### 核心模块

1. **用户认证模块**：处理用户注册、登录、登出等认证功能
2. **角色管理模块**：管理游戏角色的创建和数据同步
3. **背包系统模块**：管理角色物品的存储和使用
4. **好友系统模块**：处理好友关系的建立和管理
5. **商城系统模块**：提供商品购买功能
6. **任务系统模块**：管理任务的发布、进度跟踪和奖励发放
7. **邮件系统模块**：处理游戏内邮件的发送和接收
8. **活动系统模块**：管理游戏活动的发布和参与
9. **成就系统模块**：跟踪玩家成就的达成情况

## 8. 安全性考虑

### 前端安全

1. **输入验证**：所有用户输入必须经过验证，防止XSS攻击
2. **API调用安全**：使用HTTPS协议，防止数据传输被窃取
3. **本地存储**：敏感信息不存储在localStorage中，使用sessionStorage
4. **CORS配置**：前端API调用需要正确配置CORS头

### 后端安全

1. **密码加密**：使用BCrypt对密码进行加密存储
2. **JWT认证**：使用JWT进行身份验证，设置合理的过期时间
3. **权限控制**：实现细粒度的权限控制，防止未授权访问
4. **SQL注入防护**：使用JPA和参数化查询，防止SQL注入
5. **CSRF防护**：实现CSRF令牌验证
6. **敏感信息保护**：不在响应中返回敏感信息
7. **错误处理**：统一错误处理，不暴露系统内部错误信息
