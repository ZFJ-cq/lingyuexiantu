## 二、核心系统需求

### 1. 境界系统

#### 1.1 境界架构
| 世界层级 | 境界序列 | 核心解锁内容 |
|---------|----------|-------------|
| 人间界 | 炼体→炼气→筑基→金丹→元婴→化神→炼虚→合体→大乘 | 对应九州区域地图、基础职业、洞府系统 |
| 灵界 | 渡劫→人仙→真仙→地仙→天仙 | 灵域地图、世家阵营、灵界专属功法 |
| 仙界 | 金仙→玄仙→神仙→神王 / 至尊 | 九重天地图、天庭玩法、终极法则争夺 |

**注：每境分初 / 中 / 后 / 圆满四个阶段**

#### 1.2 突破规则
- **突破条件**：当前境界修为圆满
- **成功率公式**：基础成功率（60%）+ 灵力加成（(灵力-10)×2%）+ 神识加成（(神识-10)×1.5%）+ 气运加成（(气运-5)×3%），最高 95%
- **失败惩罚**：随机触发两种惩罚之一：1. 损失当前境界 30% 修为；2. 走火入魔，气血降至 30%
- **境界进阶**：初→中→后→下一个大境界（如炼气后→筑基）
- **天劫触发**：金丹→元婴（四九天劫 36 道）、化神→炼虚（六九天劫 54 道）、大乘→渡劫（混元天劫：心魔 + 天雷 + 业火）

### 2. 三界探索系统

#### 2.1 人间界
- **地域**：东华洲、南离洲、西极洲、北冥洲、中神州、四海（保留原有地域特性）
- **核心玩法**：季节天气 Buff（如冬季北冥洲修炼 + 20%）、秘境探索、王朝 / 宗门交互
- **解锁条件**：按境界逐步开放（炼气解锁东华洲，筑基解锁南离洲等）

#### 2.2 灵界
- **核心玩法**：三千灵域争夺、三大世家阵营选择（南宫 / 慕容 / 百里）、四大仙门交互
- **阵营特权**：选择世家后解锁专属功法（如南宫天工炼器术）、专属资源产出
- **解锁条件**：大乘圆满渡劫成功后飞升

#### 2.3 仙界
- **核心玩法**：九重天闯关（每重天需通过界主考验）、天庭任务、鸿蒙紫气收集（9 道解锁至尊神）
- **解锁条件**：灵界天仙圆满后飞升

### 3. 职业系统（四大生活职业）

#### 3.1 职业差异化设计
| 职业 | 核心定位 | 独有玩法 | 终极技能 |
|------|---------|----------|----------|
| 炼丹师 | 爆发 / 恢复 / 渡劫辅助 | 火候 QTE、丹毒系统、丹药组合 | 九转还魂丹（复活队友 + 满血满蓝） |
| 炼器师 | 战力 / 法宝 / 器灵养成 | 本命法宝成长、器灵羁绊、法宝淬炼 | 混沌开天器（全屏 AOE + 破防 50%） |
| 阵法师 | PVP / 防守 / 秘境通关 | 可移动阵法、阵眼破解、多阵组合 | 诛仙灭神阵（困住敌方 + 持续掉血） |
| 符箓师 | 应急 / 秒伤 / 保命 | 符阵连招、一次性爆发符、替身符 | 鸿蒙护体符（免疫 3 次致命伤害） |

#### 3.2 职业进阶
- **进阶路径**：九品学徒→八品→…→一品→宗师→圣手→道祖
- **进阶条件**：熟练度达标 + 制作对应品级作品 + 通过职业考核（宗师及以上需触发奇遇）
- **宗师特权**：自创丹方 / 器谱 / 阵图 / 符法、收徒传授、作品概率附加 "灵韵" 效果

### 4. 社交羁绊系统

#### 4.1 宗门系统
- **宗门等级**：九流→八流→…→一流→圣地
- **职位体系**：外门弟子→内门弟子→核心弟子→执事→长老→护法→副宗主→宗主
- **核心玩法**：宗门任务、护山大阵、宗门战争、资源共享（藏宝阁 / 藏器阁等）
- **社交联动**：宗门联赛（每年 1 次，5v5 团队战）、同门论道、长老传功

#### 4.2 道侣系统
- **结识途径**：任务邂逅、坊市相遇、宗门介绍、秘境同行
- **结为条件**：双方好感度≥80、完成 "三书六礼" 任务链、举办双修大典（消耗功德）
- **羁绊效果**：同洞府修炼速度 + 30%、解锁合击技、突破时护法成功率 + 10%、概率生育天赋后代
- **关系维护**：日常互动提升好感度，长期不互动好感度下降（最低 30）

#### 4.3 灵宠系统
- **获取途径**：主线任务、灵兽节活动、抽卡系统（保底机制）、打怪掉落宠物蛋
- **养成体系**：等级进化（50 级用化形丹进化）、技能学习、亲密度养成、外观定制
- **战斗联动**：最多 3 只上阵，亲密度≥90 解锁合体技（如九尾天狐 + 剑修 = 九尾剑域）
- **亲密度影响**：随机偷懒、30-60 普通辅助、60-90 技能联动、≥90 合体技

#### 4.4 收徒 / 孩童系统
- **收徒条件**：元婴期及以上、消耗 100 中品灵石、拥有基础功法
- **师徒福利**：徒弟修为共享 5% 给师父、师父获得功德值、解锁 "名师" 称号
- **孩童养成**：道侣生育（概率极低）、收养凡人孩童→传授功法→成为弟子→势力扩展

### 5. 修炼与功法系统

#### 5.1 修为产出
- **核心公式**：每小时修为 = 基础修为 × 灵根加成 × 体质加成 × 功法加成 × 环境加成 × 丹药加成 × 洞府加成
- **基础修为**：炼体 10 / 小时、炼气 30 / 小时、筑基 90 / 小时、金丹 270 / 小时（每大境界 ×3）
- **离线收益**：最多累积 48 小时，效率为在线的 60%（VIP 可延长至 72 小时）

#### 5.2 功法体系
- **获取途径**：宗门兑换、坊市购买、秘境探索、任务奖励、奇遇传承、自创（悟性≥100 + 化境熟练度 + 天材地宝）
- **熟练度等级**：入门→小成→大成→圆满→化境
- **功法分支**：剑修（快剑 / 重剑 / 御剑）、符箓师（雷符 / 防御符）、阵法师（杀阵 / 辅助阵）、炼丹师（攻击丹 / 辅助丹）

### 6. 商业化系统（合规无数值付费）

#### 6.1 付费类型
| 付费模块 | 内容 | 定价 | 核心价值 |
|---------|------|------|----------|
| 外观类 | 时装、飞剑皮肤、洞府皮肤、头像框、称号 | 6-198 元 | 视觉展示，无属性加成 |
| 便利功能类 | 月卡（30 元）、季卡（78 元）、年卡（288 元） | 30-288 元 | 离线收益延长、背包扩展、免费传送 |
| 资源类 | 小 / 中 / 大资源包（灵石 + 材料） | 6-328 元 | 加速养成，每日 / 周 / 月限购 |

#### 6.2 促销活动
- **新人优惠**：首充送专属飞剑、68 元成长基金（返还 688 元资源）、1 元新手礼包
- **节日促销**：春节外观 5 折、国庆双倍充值、周年庆免费送稀有外观
- **回归福利**：离线收益补偿、专属任务、回归礼包（按离开时间判定）

### 7. 任务与叙事系统

#### 7.1 任务类型
- **主线任务**：100 万字文本，按境界分卷，推动核心剧情（身世之谜→三界秘辛→天道抉择）
- **支线任务**：每个区域 10 条精品支线，丰富世界观，影响因果值
- **日常 / 周常任务**：基础资源产出，每日 / 周刷新
- **隐藏任务**：无明确提示，通过地域探索、NPC 互动触发
- **限时任务**：节日 / 活动期间开放（如灵兽节、天道赐福）

#### 7.2 叙事结构
- **第一层**：主线剧情（命运之线）→ 5 个结局（飞升 / 陨落 / 转世 / 隐居 / 改天换地）
- **第二层**：支线剧情（众生百态）→ 影响世界观评价
- **第三层**：随机事件（命运无常）→ 按气运值触发，选择影响后续发展

## 三、非核心系统需求

### 1. 洞府系统
- **核心功能**：灵田种植（产出草药 / 灵石）、修炼加速（按洞府等级 + 10%-50%）、藏经阁（解锁功法）、膜拜室（提升信仰值）
- **升级路径**：凡人洞府→灵犀洞府→仙府→天宫（消耗灵石 + 材料）

### 2. 坐骑系统
- **获取途径**：主线任务（20 级解锁青鸾）、活动兑换、抽卡系统（保底 50 抽）
- **核心价值**：移动速度 ×2、探索特殊区域、外观展示
- **养成**：等级提升、技能学习（如飞行、避障）、亲密度养成、皮肤定制

### 3. 交易系统
- **交易类型**：NPC 商店、玩家交易（以物易物 + 锁定机制）、拍卖行（跨服，5% 手续费）、坊市摆摊
- **货币体系**：1 极品灵石 = 100 上品 = 10000 中品 = 1000000 下品（绑定修仙行为消耗）

### 4. 新手引导系统
- **引导流程**：创角→灵根体质选择→移动 / 战斗 / 修炼教学→宗门选择→职业解锁→第一次突破→灵宠获取（15 分钟内完成）
- **进阶引导**：按需触发（如达到炼气期解锁修炼系统，获得第一个技能解锁技能系统）

## 四、交互设计要求

### 1. 核心界面交互
- **修炼界面**：选择功法→设置自动修炼参数（自动用药、灵力阈值）→开始修炼→实时显示进度 / 加成 / 预计突破时间
- **战斗界面**：先攻判定→我方回合（攻击 / 防御 / 物品 / 逃跑）→敌方回合→战斗结算（胜利 / 失败 / 逃跑 / 平局）
- **任务界面**：分类筛选（主线 / 支线 / 日常）→查看详情→接取→追踪进度→完成交付→领取奖励
- **社交界面**：好友 / 宗门 / 道侣分类→互动功能（传音 / 拜访 / 赠送 / 双修）→关系维护

### 2. 操作体验要求
- **轻量化操作**：核心功能 1-2 步直达，避免复杂流程
- **自动功能**：支持自动修炼、自动战斗、一键完成日常（VIP 解锁）
- **反馈机制**：突破成功、奇遇触发、战斗胜利等关键节点提供文字 + 音效反馈

## 五、性能与合规要求

### 1. 性能要求
- 安装包体积≤200MB
- 运行内存≤512MB
- 离线收益计算准确，无数据丢失
- 服务器支持万人同时在线，无卡顿

### 2. 合规要求
- 无数值付费，所有付费内容不影响游戏平衡
- 未成年人保护：设置防沉迷系统（时长限制 + 付费限制）
- 内容合规：无血腥暴力、封建迷信违规内容，符合国家手游审核标准
