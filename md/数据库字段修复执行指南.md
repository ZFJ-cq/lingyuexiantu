### 步骤 4: 检查关键指标

```sql
-- 检查字段数量
SELECT TABLE_NAME, COUNT(*) AS column_count 
FROM information_schema.COLUMNS 
WHERE TABLE_SCHEMA = DATABASE() 
AND TABLE_NAME IN ('game_role', 'role_skill', 'inventory', 'clans', 'clan_member')
GROUP BY TABLE_NAME;

-- 检查 NULL 值
SELECT 'game_role' AS table_name, 
       COUNT(*) AS total_rows,
       SUM(CASE WHEN realm_stage IS NULL THEN 1 ELSE 0 END) AS null_realm_stage
FROM game_role;

-- 检查索引
SHOW INDEX FROM game_role;
SHOW INDEX FROM role_skill;
```
