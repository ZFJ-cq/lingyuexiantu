## ⚠️ 常见问题排查

### 问题 1：手机访问显示"无法打开页面"
**原因**：服务未启动或防火墙阻止

**解决方案**：
1. 检查服务状态：
   ```bash
   lsof -i :5502  # 前端
   lsof -i :8088  # 后端
   ```
2. 关闭 macOS 防火墙：
   - 系统偏好设置 → 安全性与隐私 → 防火墙 → 关闭
3. 确保手机和电脑在同一 WiFi

### 问题 2：前端能访问但没有数据
**原因**：API 地址配置错误或 CORS 问题

**解决方案**：
1. 访问诊断页面：`http://192.168.110.98:5502/diagnose-network.html`
2. 运行完整诊断
3. 查看诊断报告中的错误信息

### 问题 3：CORS 错误
**原因**：跨域配置未生效

**解决方案**：
1. 清除手机浏览器缓存
2. 重启后端服务
3. 检查 GlobalCorsConfig.java 配置

### 问题 4：数据库连接失败
**原因**：MySQL 未启动

**解决方案**：
```bash
# 启动 MySQL
brew services start mysql

# 检查状态
brew services list | grep mysql
```
