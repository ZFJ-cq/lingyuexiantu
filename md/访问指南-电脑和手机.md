## ⚠️ 常见错误

### 错误 1：在电脑上使用局域网 IP
```
❌ 错误：http://192.168.110.98:5502 （在电脑上访问）
✅ 正确：http://localhost:5502
```

**问题**：浏览器会尝试连接网络设备，导致权限弹窗或连接失败。

**解决**：在电脑上始终使用 `localhost:5502`

### 错误 2：在手机上使用 localhost
```
❌ 错误：http://localhost:5502 （在手机上访问）
✅ 正确：http://192.168.110.98:5502
```

**问题**：localhost 指向手机本身，不是你的电脑。

**解决**：在手机上始终使用电脑的局域网 IP

### 错误 3：浏览器缓存导致的问题
```
❌ 页面显示旧版本或 CORS 错误
✅ 清除缓存后重试
```

**解决**：
- Safari：设置 → Safari → 清除历史记录与网站数据
- Chrome：设置 → 隐私 → 清除浏览数据
- Edge：设置 → 隐私 → 清除浏览数据
